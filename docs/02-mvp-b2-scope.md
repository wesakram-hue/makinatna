# MVP-B2 Scope (What ships by 2026-03-22)

## Core marketplace (end-to-end)
- Buyer: create RFQ (web form), view offers, award job
- Supplier: receive RFQs, submit offers, view awarded jobs
- Admin: supplier verification, moderation, audit logs

## Deal artifacts (anti-bypass)
- Bilingual quote/terms template
- Mobilisation checklist
- Completion proof template
- Export/share capability

## Execution tracking
- Job status timeline
- Basic dispute flag + audit trail

## Maintenance (v1)
- Maintenance plans per asset
- Due/overdue tasks
- Service logs (uploads)

## Telematics (Tier 1)
- Driver ping link (mobile) sends GPS + optional hours
- Asset last state displayed

## Ops automation (n8n self-host)
- New RFQ notification
- No-response SLA escalation
- Offer sent notification
- Award/booking confirmation notification
- Daily maintenance digest

## B2 language/RTL requirements
- /ar and /en routes
- RTL support on /ar
- Bilingual templates + notifications

## Explicit exclusions (MVP)
- Escrow/online checkout
- Public supplier pages with contact details
- Reviews/ratings
- OEM telematics integrations
